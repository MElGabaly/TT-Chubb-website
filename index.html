Please follow all the instructions below and it is important to start the choices from top to bottom as they are listed below.
Please choose one of the tasks from the dropdown menu that is most relevent to the work you have done. once the task has been chosen some fields will appear to fill, be as specific as possible and to the point when answering them.
Please choose the most relevent activity to the task you chose
When you finish filling in all the field, press on "Generate Comment" Button, and the comment will appear in the textbox shown below. Then press on copy comment (An Alert is going to appear on the top indicating the comment has been copied to the clipboard), so you can past it in deltek. Finally, press on generate new comment, which will empty all the field so you can start a new comment
